## 用户
###### 用户注册  测试 ok
api:/api/user/register  
method： post
参数：userInfo(obj)
必含参数：userName / password / tel / email / status(0:正常 / 1:删除) / mailAddress / roleType (0:管理员 / 1:Agent / 2:房东 / 3: 房客)
返回值： code   message   data
